Properties

ForceUpdates
Controls whether Time of Day updates take effect immediately.
Hour
The hour value for the Time of Day system. The value is wrapped, so setting the value to 24 will reset the hour to zero.
Minute
The minute value for the Time of Day system. The value is wrapped, so setting the value to 60 will increment the hour and reset the minutes to zero.
Speed
Controls the speed at which the Time of Day passes.
