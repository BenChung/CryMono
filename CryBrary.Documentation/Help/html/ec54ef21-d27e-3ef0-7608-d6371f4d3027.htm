The PathUtils type exposes the following members.
Properties
  NameDescription
BinaryFolder
Gets the current folder containing binaries relevant to current build type (Bin32 / Bin64).
EngineFolder
Gets the engine folder. CryENGINE uses this store engine content which is not specific to the game being developed, i.e. shaders and configuration files.
GameFolder
Gets the current game folder. CryENGINE uses this as the root for all game-related content.
ScriptsFolder
Gets the folder in which the engine stores scripts. We use this as the root for our .cs files and .dll plugins.
