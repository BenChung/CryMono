Description

CryMono brings the power of .NET into the world of CryENGINE3, allowing for game logic to be scripted in a much faster and more intuitive way than before.

Non-commercial projects may use CryMono free-of-charge. For other projects; contact us at business@inkdev.net.

Documentation

For instructions on all things CryMono, see our Confluence setup at docs.inkdev.net.

Source directory structure

Our Visual Studio projects have been set up to expect all contents to be placed inside a folder within the Code folder shipped with the CryENGINE Free SDK.

Example: D:\Dev\INK\CryENGINE\Code\CryMono

Using another folder structure is up to the developer, but will require customization in order for reference and output locations to be correct.
